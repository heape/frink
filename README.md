Electronを使ったフリマサイトのリサーチツール

### ・使い方

このレポジトリをクローンし、npm updateでモジュールを追加
electron .
上記のコマンドで起動できます。

Web APIはPHPが実行できるサーバーをローカルホスト、またはウェブサーバー上にアップロードし、
app.jsのAPI urlをお好きなように変更してください。

### ・対応済みサイト
メルカリ、ラクマ
